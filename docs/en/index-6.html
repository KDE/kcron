6. About Cron

Please see the following manual pages for more technical information about cron and crontab, the native Unix system utilities used by KCron.

- crontab: Maintain crontab files for individual users. (Enter "man:crontab(1)" in KDE Help.)

- crontab file format: Tables for driving cron. (Enter "man:crontab(5)" in KDE Help.)

- cron: Daemon to execute scheduled commands. (Enter "man:cron(8)" in KDE Help.)
