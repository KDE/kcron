5. Environment Variables

5.1 Create New Environment Variable

To create a new environment variable, first select the "Variables" folder.

Then select Edit/New from the menu. Alternatively you can use the right mouse click menu.

You will see the "Edit Variable" dialog.

Variable

In the "Variable" field, enter the environment variable name. You can use the drop-down list box to select from the most common environment variables used by scheduled tasks. Those include:

- HOME: To be used instead of the default user's home directory

- MAILTO: To send email output to an email address instead of the the user's default email address.

- PATH: To be used to search directories for program files.

- SHELL: To be used instead of the user's default value.

Note: When you change the variable, both the icon, and the comment may be updated.

Value

In the "Value" field, enter the environment variable value.

Comment

In the "Comment" field, enter a description for the environment variable, such as its purpose.

Enabled

If you want to disable the environment variable, de-select, the "Enabled" check box.

If you want to enable the environment variable, select the "Enabled" check box.

Press the "OK" button to continue.

Note: The environment variable is not actually set up until saved. All modifications can be saved by selecting File/Save from the menu.

Alternatively press the "Cancel" button to abandon creating this environment variable.

5.2 Modify Environment Variable

To modify an environment variable, first select the "Variables" folder.

Then select Edit/Modify from the menu. Alternatively you can use the right mouse click menu.

You will see the "Edit Variable" dialog.

Variable

In the "Variable" field, change the environment variable name as necessary. You can use the drop-down list box to select from the most common environment variables used by scheduled tasks. Those include:

- HOME: To be used instead of the default user's home directory

- MAILTO: To send email output to an email address instead of the the user's default email address.

- PATH: To be used to search directories for program files.

- SHELL: To be used instead of the user's default value.

Note: When you change the variable, both the icon, and the comment may be updated.

Value

In the "Value" field, modify the environment variable value as necessary.

Comment

In the "Comment" field, enter a description for the environment variable, such as its purpose.

Enabled

If you want to disable the environment variable, de-select the "Enabled" check box.

If you want to enable the environment variable, select the "Enabled" check box.

Press the "OK" button to continue.

Note: The environment variable is not actually set up until saved. All modifications can be saved by selecting File/Save from the menu.

Alternatively press the "Cancel" button to abandon creating this environment variable.

5.3 Delete Environment Variable

To delete an environment variable, first select the variable to be deleted.

Then select Edit/Delete from the menu. Alternatively you can use the right mouse click menu.

Note: The deletion is not actually made until saved. All modifications can be saved by selecting File/Save from the menu.

5.4 Cut Environment Variable

To cut an environment variable, first select the variable to be cut.

Then select Edit/Cut from the menu. Alternatively you can use the right mouse click menu or tool bar.

Note: The deletion is not actually made until saved. All modifications can be saved by selecting File/Save from the menu.

5.5 Copy Environment Variable

To copy an environment variable, first select the variable to be copied.

Then select Edit/Copy from the menu. Alternatively you can use the right mouse click menu or tool bar.

Once an environment variable has been cut or copied, paste will be enabled.

5.6 Paste Environment Variable

To paste an environment variable, first an environment variable must have already been cut or copied to the clipboard.

Then select the "Variables" folder.

Finally select Edit/Paste from the menu. Alternatively you can use the right mouse click menu or tool bar.

Note: The newly pasted variable is not actually set up until saved. All modifications can be saved by selecting File/Save from the menu.

5.7 Enable Environment Variable

To enable an environment variable, first select the disabled variable. (A disabled variable will display "Disabled" in its description.)

Then select Edit/Enabled from the menu. Alternatively you can use the right mouse click menu.

Confirm that the environment variables's program name and parameters as well as description are displayed.

5.8 Disable Environment Variable

To disable an environment variable, first select the enabled variable. (A disabled variable will display its value and its description.)

Then select Edit/Enabled from the menu. Alternatively you can use the right mouse click menu.

Confirm that the environment variables's program name and parameters are no longer displayed and the description "Disabled" is displayed.
