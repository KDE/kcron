1. Introduction

1.1 Description

KCron is an application for scheduling programs to run in the background. It is a graphical user interface to cron, the Unix system scheduler.

1.2 Acknowledgements

KCron was initially based upon kcrontab developed by Michael Bialas and documented by David Klemm.

KCron acknowledges Paul Vixie for vixie-cron.
