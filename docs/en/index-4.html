4. Scheduled Tasks

4.1 Create New Scheduled Task

To create a new scheduled task, first select the "Tasks" folder.

Then select Edit/New from the menu. Alternatively you can use the right mouse click menu. You will see the "Edit Task" dialog.

Comment

In the "Comment" field, enter a description of the task to schedule.

Program

In the "Program" field, enter the name of the program. You can specify either a relative path or absolute path. If you want to look up the program, press the "Browse..." button.

Enabled

If you want to disable the task, de-select, the "Enabled" check box.

Daily

If you want to schedule the task to run daily, select the "Run Every Day" check box.

Months

Check the months to schedule the task.

Days of the Month

Check the days of the month to schedule the task.

Days of the Week

Check the days of the week to schedule the task.

Note: If you select both several days of the month and several days of the week, the task will run when either condition is met.

For instance, if you select the 1st and 15th and select Sunday, the program will be run every 1st and 15th of the selected months (regardless of day of week) as well as every Sunday of the selected months (regardless of day of the month.

To run on a weekday schedule, check all days of the month and the selected days of the week.

To run on a days of the month schedule, check all days of the week, and the selected days of the month.

Hours

Check the hours to schedule the task.

Minutes

Check the minutes to schedule the task. (KCron does not support scheduling tasks at more granular than five minute intervals.)

Press the "OK" button to continue.

Note: The scheduled task is not actually set up until saved. All modifications can be saved by selecting File/Save from the menu.

Alternatively press the "Cancel" button to abandon creating this new task.

4.2 Modify Scheduled Task

To modify a scheduled task, first select the task to be modified. Then select Edit/Modify from the menu. Alternatively you can use the right mouse click menu. You will see the "Edit Task" dialog.

Comment

In the "Comment" field, modify the description as needed.

Program

In the "Program" field, modify the name of the program as needed. You can specify either a relative path or absolute path. If you want to look up the program, press the "Browse..." button.

Enabled

If you want to enable the task, select, the "Enabled" check box.

If you want to disable the task, de-select, the "Enabled" check box.

Daily

If you want to schedule the task to run daily, select the "Run Every Day" check box.

Months

Check the months to schedule the task.

Days of the Month

Check the days of the month to schedule the task.

Days of the Week

Check the days of the week to schedule the task.

Note: If you select both several days of the month and several days of the week, the task will run when either condition is met.

For instance, if you select the 1st and 15th and select Sunday, the program will be run every 1st and 15th of the selected months (regardless of day of week) as well as every Sunday of the selected months (regardless of day of the month.

To run on a weekday schedule, check all days of the month and the selected days of the week.

To run on a days of the month schedule, check all days of the week, and the selected days of the month.

Hours

Check the hours to schedule the task.

Minutes

Check the minutes to schedule the task. (KCron does not support scheduling tasks at more granular than five minute intervals.)

Press the "OK" button to continue.

Note: The modification is not actually made until saved. All modifications can be saved by selecting File/Save from the menu.

Alternatively press the "Cancel" button to abandon modifying this scheduled task.

4.3 Delete Scheduled Task

To delete a scheduled task, first select the task to be deleted.

Then select Edit/Delete from the menu. Alternatively you can use the right mouse click menu.

Note: The deletion is not actually made until saved. All modifications can be saved by selecting File/Save from the menu.

4.4 Cut Scheduled Task

To cut a scheduled task, first select the task to be cut. Then select Edit/Cut from the menu. Alternatively you can use the right mouse click menu or tool bar.

Note: The deletion is not actually made until saved. All modifications can be saved by selecting File/Save from the menu.

4.5 Copy Scheduled Task

To copy a scheduled task, first select the task to be copied.

Then select Edit/Copy from the menu. Alternatively you can use the right mouse click menu or tool bar.

Once a scheduled task has been cut or copied, paste will be enabled.

4.6 Paste Scheduled Task

To paste a scheduled task, first a scheduled task must have already been cut or copied to the clipboard.

Then select the "Tasks" folder. Then Finally select Edit/Paste from the menu. Alternatively you can use the right mouse click menu or tool bar.

Note: The newly pasted task is not actually set up until saved. All modifications can be saved by selecting File/Save from the menu.

4.7 Enable Scheduled Task

To enable a scheduled task, first select the disabled task. (A disabled task will display "Disabled" in its description.)

Then select Edit/Enabled from the menu. Alternatively you can use the right mouse click menu.

Confirm that the schedule task's program name and parameters as well as description are displayed.

4.8 Disable Scheduled Task

To disable a scheduled task, first select the enabled task.

Then select Edit/Enabled from the menu. Alternatively you can use the right mouse click menu.

Confirm that the schedule task's program name and parameters are no longer displayed and the description "Disabled" is displayed.

4.9 Run Scheduled Task Now

To run a scheduled task immediately, first select the task.

Then select Edit/Run Now from the menu. Alternatively you can use the right mouse click menu.
