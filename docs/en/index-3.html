3. Onscreen Fundamentals

3.1 Start Up

When KCron starts you will see a summarized view of scheduled tasks and associated environment variables. If you are running as the root user, you will see these items for all users on the computer as well as the system scheduled tasks. Each of the folders can be expanded and contracted.

Scheduled Tasks

Scheduled tasks appear under a "Tasks" folder. For each scheduled task, the following are displayed:

- Name

- Value: Program file and parameters

- Description: Natural language description of schedule

If a task has been disabled, no program file and parameters will appear and the description will be "Disabled."

Environment Variables

Environment various appear under a "Variables" folder. For each environment variable, the following are displayed:

- Name

- Value

- Description

If an environment variable has been disabled, no value will appear and the description will be "Disabled."

3.2 Save

To save any modifications you have made, select File/Save from the menu. Alternatively you call press the save icon on the toolbar.

If you have made any modifications, the program will prompt you to save before exiting.

Note: No changes are actually made until you explicitly save from the menu or toolbar, or when exiting.

3.3 Quit

To quit the program, select File/Quit from the menu.

If you have made any modifications and have not saved them, the program will prompt you to save before exiting.

You will be asked, "Scheduled tasks have been modified. Do you want to save changes?"

- Answer "Yes" to save changes and exit.

- Answer "No" to abandon changes and exit.

- Answer "Cancel" to continue editing.

3.4 Show Tool Bar

If the tool bar is not visible, select Options/Show Tool Bar from the menu. The tool bar will be displayed and the Show Tool Bar menu item will be checked.

3.5 Hide Tool Bar

If the tool bar is visible, select Options/Show Tool Bar from the menu. The tool bar will no longer be displayed and the Show Tool Bar menu item will be unchecked.

3.6 Show Status Bar

If the status bar is not visible, select Options/Show Status Bar from the menu. The status bar will be displayed and the Show Status Bar menu item will be checked.

3.7 Hide Status Bar

If the status bar is visible, select Options/Show Status Bar from the menu. The status bar will no longer be displayed and the Show Status Bar menu item will be unchecked.
